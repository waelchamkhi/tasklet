# Tasklet 🧩
A lightweight, privacy-first to-do list that runs entirely in your browser.
No accounts. No tracking. No unnecessary complexity.
---
## ✨ Features
* ✅ Create, edit, and delete tasks
* 📌 Mark tasks as complete
* 🏷️ Color Coded Tasks 
* 💾 Local storage persistence
* 📤 Export tasks to JSON
* 📥 Import tasks from JSON
* 🌙 Dark mode
---
## 🧠 Philosophy
Tasklet is designed to be:
* **Fast** – no backend, instant load
* **Private** – your data stays in your browser
* **Portable** – export/import anytime
* **Simple** – focused on what actually matters
